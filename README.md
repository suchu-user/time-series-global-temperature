# Time-Series Analysis of Global Temperature Data

Summer Internship Project 2026 — IDEAS (Institute of Data Engineering, Analytics and Science Foundation), ISI Kolkata.

**Author:** Suchethan PH
**Period:** 18 May 2026 – 14 June 2026

## Overview
Exploratory time-series analysis of monthly global surface-temperature anomalies
(1880–2016, sources GCAG and GISTEMP), followed by a K-Means clustering exercise
on the scikit-learn digits dataset evaluated with a macro-averaged F1 score.

## Contents
| File | Description |
|------|-------------|
| `Time_Series_Analysis_Global_Temperature_Summer_2026.ipynb` | Completed project notebook (Questions 1–13) |
| `monthly_csv.csv` | Global temperature anomaly dataset used by Q1–Q10 |
| `Summer_Internship_Report_2026.pdf` | Project report |

## How to run
```bash
pip install pandas numpy scikit-learn
jupyter notebook Time_Series_Analysis_Global_Temperature_Summer_2026.ipynb
```
Run the cells top to bottom. The notebook loads `monthly_csv.csv` from the same folder.

## Tools
Python 3, pandas, numpy, scikit-learn.
